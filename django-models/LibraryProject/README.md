# LibraryProject

A Django-based library management system developed as part of the ALX Django Learning Lab.

## Description

This project is a library management system built with Django framework.

## Setup Instructions

1. Install Django:
```
   pip install django
```

2. Run migrations:
```
   python manage.py migrate
```

3. Start the development server:
```
   python manage.py runserver
```

## Features

- Library management functionality
- Built with Django 6.0
- SQLite database
